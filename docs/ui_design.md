## 一、整体布局

### 1.1 页面结构

采用标准的顶部导航 + 主内容区域布局：

**顶部导航栏**：固定在顶部，包含Logo、主要入口导航、功能菜单、设置和用户头像。

**主内容区域**：根据不同页面展示对应内容，支持滚动。

### 1.2 导航结构

**Logo区域**：左侧显示AutoVid品牌标识。

**主要入口**：
- 短剧创作
- 视频生成

**功能菜单**：
- 我的项目
- 角色库
- 模板中心

**右侧区域**：
- 设置按钮
- 用户头像（下拉菜单：个人设置、退出登录）

## 二、首页设计

### 2.1 入口选择页

**页面标题**：欢迎使用 AutoVid

**副标题**：AI视频创作平台，让创意一键成片

**双入口卡片**：

**入口一：短剧创作**
- 图标：电影胶片图标
- 功能说明：文字 → 剧本 → 角色 → 分镜 → 完整短剧
- 适用场景：连续剧、系列内容
- 操作按钮：「开始创作」

**入口二：视频生成**
- 图标：摄像机图标
- 功能说明：人物 → 场景 → 视频，保持角色一致性
- 适用场景：单条视频、角色IP打造
- 操作按钮：「开始生成」

**近期项目区域**：
- 标题：「近期项目」
- 展示形式：横向排列的项目卡片
- 卡片内容：封面图、项目名称、更新时间
- 点击卡片进入对应项目

| 字段 | 控件类型 | 选项 |

## 八、交互规范

### 8.1 提示词编辑交互

**触发时机**：点击「生成」相关按钮时

**交互流程**：

1. 弹出提示词设置面板（侧边滑出或模态弹窗）
2. 面板显示：
   - 默认提示词模板（折叠状态，可展开查看）
   - 补充提示词输入框
   - 说明文字
3. 用户可：
   - 查看默认模板
   - 在补充框输入自定义内容
   - 点击「跳过，使用默认」直接生成
   - 点击「应用并生成」使用自定义提示词

**拼接逻辑**：最终提示词 = 默认模板 + 用户补充 + 用户输入

### 8.2 模型切换交互

**触发**：下拉选择模型提供商

**交互流程**：

1. 显示已配置的提供商列表
2. 未配置的提供商显示警告图标和「未配置」
3. 选择后，下方表单动态显示该提供商的配置项
4. 已配置的提供商显示绿色勾选标记

### 8.3 图片选择交互

**触发**：生成角色形象后

**交互流程**：

1. 图片网格展示
2. 鼠标悬停显示放大图标
3. 点击图片切换选中状态
4. 选中的图片显示边框高亮和勾选标记
5. 可多选（如果业务需要）或单选
6. 底部显示当前选中数量

### 8.4 视频生成进度

**触发**：点击「生成视频」后

**交互流程**：

1. 显示全屏遮罩或进度面板
2. 进度条显示当前进度
3. 显示预计剩余时间
4. 显示当前步骤文字说明
5. 提供取消按钮
6. 完成后自动跳转到结果页或关闭进度面板

### 8.5 发布进度

**触发**：点击「发布」后

**交互流程**：

1. 显示每个平台的发布状态
2. 发布成功显示绿色勾
3. 发布失败显示红色叉和错误原因
4. 全部完成后显示成功提示
5. 提供查看已发布内容的链接
